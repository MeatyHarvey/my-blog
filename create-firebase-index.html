Manual Index Creation

If the automatic method doesn't work, you can manually create the index in Firebase Console:

  1. Go to Firebase Console - Indexes
  2. Click "Create Index"
  3. Collection ID: comments
  4. Add these fields:
    • Field: postId, Order: Ascending
    • Field: timestamp, Order: Descending
  5. Click "Create"
